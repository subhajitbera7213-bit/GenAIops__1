# 📖 Overview

This project demonstrates a complete **GenAIOps workflow** for developing, evaluating, and monitoring an AI agent using **Azure AI Foundry**.

Instead of simply building an AI chatbot, this project follows a production-oriented lifecycle:

- AI Agent Development
- Prompt Engineering
- GitHub Actions CI/CD
- Automated AI Evaluation
- Azure AI Foundry Evaluation
- AI Monitoring
- Distributed Tracing
- Continuous Quality Validation

# 🔄 CI/CD Workflow

This project integrates **GitHub Actions** with Azure AI Foundry.

Workflow includes:

- Checkout Repository
- Install Dependencies
- Azure Login
- Execute Agent Evaluation
- Generate Evaluation Results
- Comment Results on Pull Request

This enables automated AI quality checks before merging code.

---

# 📊 Automated AI Evaluation

The Trail Guide Agent is evaluated using Azure AI Foundry across multiple quality dimensions.

### Evaluation Metrics

- ✅ Intent Resolution
- ✅ Relevance
- ✅ Groundedness

Results are automatically generated and attached to Pull Requests.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/subhajitbera7213-bit/GenAIops__1.git
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Azure

Login using Azure CLI

```bash
az login
```

Configure your Azure AI Foundry credentials.

---

## Run Evaluation

```bash
python src/evaluators/evaluate_agent.py
```
